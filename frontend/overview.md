# 📘 Project Overview – GigShield AI

GigShield AI is designed to provide **income protection** for gig workers using a **parametric insurance model**.

## 👤 Target User

Delivery partners working for platforms like:
- Zomato
- Swiggy
- Zepto

## 📌 Key Idea

Instead of manual claims, the system:
- Monitors external disruptions
- Automatically triggers payouts
- Uses AI for risk and fraud detection

## 🔄 Workflow Summary

1. User onboarding  
2. Premium calculation  
3. Policy activation  
4. Real-time monitoring  
5. Trigger detection  
6. Automatic payout  

## 🎯 Objective

To ensure **financial stability for gig workers** during unpredictable events.
